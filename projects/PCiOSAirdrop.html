June 2, 2025

When working on things I usually like to write things on my iPad (like state diagrams for the FPGALLM stuff). But, I also like referring to this stuff on my PC. For example, if I wanted to include it here, I need it on my PC. My default has been sharing it through Google Drive, but I don't like that. When I used to use a mac (I had to swap to windows for vivado, but now that I'll be working on EC2 it might be time to swap back), airdrop was a life saver. I didn't think I was the first person to want this, but I couldn't find a way to do it online that is as seamless as airdrop. I haven't done anything yet, but I'm putting this here so I don't procrastinate (as much as I would otherwise).
