It's been a while since I've written anything in Swift so it took me a little bit to get back into it. But, it works. I didn't realise how many keys were actually mapped to shortcuts, so the most frustrating part was figuring out which ones I could use. I've now trippled my clipboard size. I can use command + j to copy something, and command + shift + j to paste it. Same for m. There's still some stuff that I haven't figured out. 1. Whenever I open the app it shows me the permissions view, I know why but I don't know how to fix it yet (I ran into some syntax errors with my fix) 2. I imagine this is helpful to run in the background, so how do I do that? 3. I feel like it should allow you to pick your key combinations. For example, what if someone else has m and j mapped to things, they should be able to use control or something else. I can add these in along the way, but for now it works. Also a quick note, the command + c and command + v keys shown after the j and m ones are simulated keypresses, and command + c/v works the same.
